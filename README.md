# 🎉 網址提取器 v2.0 更新內容
## 本次更新對網址提取器進行了大幅度的功能擴充與優化，讓工具變得更加強大與實用。

### 主要更新亮點：

#### 自動移除重複網址與數量顯示

現在，工具會自動辨識並移除所有重複的網址，並在提取完成後，在下方訊息區顯示「移除了多少個重複項目」，讓您對資料的重複性一目瞭然。

#### 新增「寬鬆模式」

除了原本只提取 http:// 或 https:// 開頭的嚴格模式，我新增了「寬鬆模式」選項。勾選後，工具將能辨識並提取更多形式的網址，例如：www.google.com 。


#### 即時網址過濾器
提取完網址後，會出現一個新的過濾輸入框。您可以在這裡輸入任何關鍵字，工具會即時篩選出包含該關鍵字的網址，幫助您更快找到需要的內容。

#### 自動移除重複網址與數量顯示
現在，工具會自動辨識並移除所有重複的網址，並在提取完成後，在下方訊息區顯示「移除了多少個重複項目」，讓您對資料的重複性一目瞭然。

#### 一鍵下載所有網址
新增了「下載網址」按鈕。點擊後，所有提取出的網址將會被儲存為一個名為 <code>extracted_urls.txt</code> 的文字檔案，方便您離線使用或備份。
